# Contributing to DigiCode

DigiCode is currently released as open source under AGPL-3.0 for transparency and AGPL §13 compliance. However, **we do not currently accept external contributions** including but not limited to:

- Pull Requests
- Issues
- Bug reports
- Feature requests

## Why?

DigiCode is developed and maintained by DigiCo LLC as part of FabLab West-Harima's educational programs. We do not have the resources to manage external contributions at this time.

## Inquiries

For inquiries, please use one of the following channels:

- DigiCode users: Use the "Send Feedback" option in the HELP menu of the DigiCode application
- General inquiries: [FabLab West-Harima Contact Form](https://fablab-westharima.jp/contact)

## Future Plans

We may open contributions in the future. Updates will be announced on this repository.
